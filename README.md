# Student Portfolio Website

A personal portfolio website built from scratch using HTML and CSS, created as part of a UI/UX assignment.

## 🔗 Live Demo
[View Portfolio](https://zaheer-sza.github.io/PORTFOLIO/)

## 📋 About
Hi, I'm Syed Zaheer Ahmed, a B.Tech Computer Science student at Amrita Vidya Vishwapeetham, Vijayawada. This portfolio was built to showcase my academic background, skills, projects, hobbies, and personal information.

## 🛠️ Built With
- **HTML5** — page structure and semantic markup across multiple pages
- **CSS3** — styling, gradients, hover effects, and responsive layout
- **Font Awesome** — icons for social links and UI elements
- **Google Fonts** — Poppins typeface
- **External image CDNs** — for social media icons
## 📌 Pages Breakdown

### `interface.html` — Landing Page
- Hero layout with name, intro text, and profile card
- "About Me" button linking to the main portfolio
- Profile photo displayed in a circular avatar card
- Social icons (Instagram, Twitter, GitHub)

### `index.html` — Main Portfolio
- Profile section with photo, name, roll number, and social icon links
- Academic table showing college and B.Tech grades
- Technical skills (Python, C, C++, Java)
- Non-technical skills (journalism, badminton, photography)
- Curricular activities section
- Links to Hobbies, Personal Info, and Additional Info pages
- Footer with contact email and phone

### `personal1.html` — Personal Information
- Full personal details (DOB, city, languages, strengths)
- Personal statement and life principle quote
- Web and Python projects listed
- Physical project — Rain-Sensing Automatic Car Wiper
- Portfolio project description with image

### `hobbies1.html` — Hobbies
- Photography, Badminton, and Travel hobby cards
- Music interests (Guitar, Piano, Drums)
- Coding interests (C++, Java, Python, algorithms)

### `travel1.html` — Additional Information
- Travel timeline (Hyderabad, Bangalore, Kerala)
- Personal reflection statement
- Extracurricular activities (photography, badminton, volunteering, clubs)

## ✨ CSS Features Used
- `linear-gradient` background (black → grey → red)
- Semi-transparent section cards using `rgba`
- Hover effects with `transform: scale()` and color transitions
- Circular avatar styling with `border-radius: 50%`
- `.back-home` fixed position button on sub-pages
- Shared `style.css` across all pages for consistent styling
- Box shadows and border highlights for card elements
